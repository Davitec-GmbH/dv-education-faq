# Education FAQ (dv_education_faq)

Structured FAQ system for the education sector with category-based filtering, search, and optional helpfulness rating.

## Features

- FAQ items with rich text answers and category assignment
- Search across questions and answers
- Category filter dropdown
- "Was this helpful?" rating (Yes/No) with AJAX
- Schema.org Question/Answer markup for SEO
- TYPO3 v12 LTS and v13 LTS support

## Installation

```bash
composer require davitec/dv-education-faq
```
