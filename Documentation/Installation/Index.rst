Requirements
============

- PHP 8.2+
- TYPO3 v12.4 LTS or v13.4 LTS

Installation
============

.. code-block:: bash

   composer require davitec/dv-education-faq

Activate the extension, run database schema updates, and include the
static TypoScript template.
